安装说明

  1. 下载完成后，先解压整个 MonitorAgent.zip，不要直接在压缩包里运行。
  2. 进入解压后的文件夹，双击 install-agent.bat。
  3. 如果弹出 Windows 管理员授权窗口，请点击“是”。
  4. 安装窗口会等待安装结束，并检查后台进程和计划任务是否真的启动。
如果安装后仍显示离线，请查看日志：%LOCALAPPDATA%\Windows Monitor\logs\install.log。
如果杀毒软件拦截，请将 WindowsMonitor.exe 加入信任列表。
