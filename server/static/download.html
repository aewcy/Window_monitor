安装说明

  1. 下载完成后，双击运行 WindowsMonitorSetup.exe。
  2. 如果弹出 Windows 管理员授权窗口，点击“是”。
  3. 安装器会自动复制程序、注册后台启动，并启动 Windows Monitor。
  4. 卸载时可在开始菜单打开 Windows Monitor 分组，点击 Uninstall Windows Monitor；也可以在 Windows 设置的应用列表中卸载。
如果安装后仍显示离线，请查看日志：%LOCALAPPDATA%\Windows Monitor\logs\install.log
